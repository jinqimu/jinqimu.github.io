PREVIEW

之前有用过GitHub作为博客的图床，然而体验并不好。GitHub使用Amazon的服务器导致下载速度在国内惨不忍睹，而我又是一个摄影爱好者，常会挂载一些数兆大小的图片，久而久之也就只能弃用自建博客了。

最近由于一只蝙蝠引起的故事，闲着没事做更新了博客采用Gridea作为驱动，顺带发现了JSDeliver这个免费CDN。
